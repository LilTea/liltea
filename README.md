# LilTea

LilTea is a golfing language :)

## Getting Started

To get started you can use BigTea to write LilTea :)

### Prerequisites

You need node.js :)

```
Give examples
```

### Running

To run liltea:

```
node liltea.js liltea.js < source.lit
```

To compress bigtea:

```
compressor.js < source.lit
```
